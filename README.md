### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/Abauchot/Fit-Quests.git
   cd Fit-Quests
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**

   ```bash
   npm start
   # or
   yarn start
   ```

4. **Run on your preferred platform**

   ```bash
   # iOS
   npm run ios
   
   # Android
   npm run android
   
   # Web
   npm run web
   ```
